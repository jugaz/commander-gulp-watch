## Installation

```bash
$ npm install commander-gulp-watch
```


#### Command to Compile

```bash
$ commander-gulp-watch images 'entry' --w 'ouput' 
```

#### Example

```bash
"scripts": {
    "watch": "commander-gulp-watch watch 'test/images/**/*.png' 'test/images/**/*.jpg' --w 'build/images'"
  }
```
